UAV SAR Video Feed

↓ video frames

C1
YOLOv8 Person Detector
HERIDAL fine-tuned • conf ≥ 0.1 • ~82ms/frame

C2
ByteTrack MOT
IoU + centroid distance association

→
Filter
Track Quality
≥15px mean size
≥5 frame history
↓ validated tracks (centroids, bboxes, confidences)
Stage 2 — Multi-Stream Classification 2
Trajectory Stream A
C3
TMS-16 + Random Forest
16 hand-crafted motion features
SMOTE-balanced, 200 trees
60.3% accuracy (our contribution)
Trajectory Stream B
C4
TrajMAE
Masked autoencoder on
trajectory delta tokens
C5
Anomaly Detector
Reconstruction error → anomaly flag
Trajectory Stream C
C6
SCTE
Self-supervised Contrastive
Trajectory Encoder
+ linear probe classifier
Pixel Stream
C7
MViTv2-S
Video transformer on person crops
Kinetics-pretrained → SAR fine-tuned
26.5% at <50px (chance level)
C8
Majority Vote Ensemble
Fuses TMS-16 + TrajMAE + SCTE class predictions via majority vote → {lying_down, stationary, walking, running}
↓ ensemble class • confidence • anomaly flag
Stage 3 — Temporal & Context Analysis 3
C9
TCE v3 — Temporal Criticality Engine
Confidence-aware state machine
MOVING → STATIONARY → COLLAPSED → STOPPED
score = base × dwell_escalation × (0.5 + 0.5 × max_class_prob)
C10
EMI — Ego-Motion Intelligence
ORB features → homography decomposition
Flight phase: TRANSIT / SCANNING / HOVERING
Hovering = operator interest → boost priority
↓ TCE score • EMI attention multiplier
Stage 4 — Fusion & Ranking 4
C11
AAI-v2 Meta-Classifier
3-layer MLP → softmax weights [wpixel, wtraj]
Input: person_size, det_confidence, motion_magnitude
fused = wpixel × MViT_score + wtraj × (ensemble_score × TCE)
Ranker
Priority Ranker
final_priority = fused_score × EMI_multiplier → sort descending
↓ ranked priority list
